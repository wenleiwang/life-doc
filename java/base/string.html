# StringгҖҒStringBuilderгҖҒStringBufferзҡ„еҢәеҲ«

StringеҜ№иұЎдёҚеҸҜеҸҳпјҢStringBuilderгҖҒStringBufferеҸҜеҸҳгҖӮ StringBuilderrжү§иЎҢйҖҹеәҰжңҖдҪіпјҢStringBufferж¬Ўд№ӢпјҢStringзҡ„жү§иЎҢйҖҹеәҰжңҖж…ўпјҲStringдёәеӯ—з¬ҰдёІеёёйҮҸпјҢиҖҢStringBuilderе’ҢStringBufferеқҮдёәеӯ—з¬ҰдёІеҸҳйҮҸпјҢStringеҜ№иұЎдёҖж—ҰеҲӣе»әеҗҺиҜҘеҜ№иұЎжҳҜдёҚеҸҜжӣҙж”№зҡ„пјҢеҗҺдёӨиҖ…зҡ„еҜ№иұЎжҳҜеҸҳйҮҸжҳҜеҸҜд»Ҙжӣҙж”№зҡ„пјү StringгҖҒStringBufferжҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢStringBuilderжҳҜзәҝзЁӢдёҚе®үе…Ёзҡ„
