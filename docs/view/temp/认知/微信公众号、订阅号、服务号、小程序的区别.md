# 微信公众号、订阅号、服务号、小程序的区别


[尝试开发微信公众号消息推送功能并且和小程序关联](https://developer.aliyun.com/article/1145101)

[公众平台服务号、订阅号、企业微信以及小程序的相关说明](https://kf.qq.com/faq/120911VrYVrA130805byM32u.html)

研读如上两篇文章弄清楚了微信公众号、订阅号、服务号、小程序的区别：

公众号包含订阅号、服务号两种类型。申请注册公众号时会要求选择订阅号和服务号。==这两者都是公众号==，可以理解为服务号的功能更强大，但是需要企业级的资质才可以。

![image.png|950](https://img-life.oss-cn-beijing.aliyuncs.com/doc/202402181548855.png)


1. 订阅号：主要偏于为用户传达资讯（类似报纸杂志），认证前后都是每天只可以群发一条消息；
2. 服务号：主要偏于服务交互（类似银行，114，提供服务查询），认证前后都是每个月可群发4条消息；
3. 企业微信：是一个面向企业级市场的产品，是一个独立的APP好用的基础办公沟通工具，拥有最基础和最实用的功能服务，专门提供给企业使用的IM产品。（适用于企业、政府、事业单位或其他组织）；
4. 小程序：是一种新的开放能力，开发者可以快速的开发一个小程序。小程序可以在微信内被便捷的获取和传播，同时拥有出色的使用体验。
